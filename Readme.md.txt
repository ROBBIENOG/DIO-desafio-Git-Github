Desafio DIO

Criação de repositório para treinar uso de comandos no Git e Github.

1 - Criação de pasta no computador para salvar arquivos do desafio;
2 - Criação de arquivo Readme.md;
3 - Abrir pasta no Git Bash;
4 - Usar comandos: 
* git init (para iniciar um repositório vazio na pasta selecionada)
* git status (verificar o status dos arquivos na pasta)
* git add . (add os arquivos para serem comitados)
* git commit -m "initial commit DIO desafio" (preparar o arquivo para enviar pro Github)
* git push origin ssh (para empurrar pro Github).
